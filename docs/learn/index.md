# About

This section is a tutorial and guides you through the basic steps required to build your own Datashare worker **from scratch**.

**In practice, you won't have to start anything from scratch**.
The [get started](../get-started/index.md) section details how to create a Datashare worker by cloning the [datashare-python](https://github.com/ICIJ/datashare-python) template repo.
